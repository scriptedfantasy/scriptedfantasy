## Technical Requirements

- **Agentic readability:** All content in DOM (not hidden behind JS). Schema.org JSON-LD. `llms.txt`. Semantic HTML5. Clean text-to-markup ratio.
- **Mobile-first:** Responsive CSS, stacking on small screens.
- **Local media:** All images/videos stored in `assets/media/`. YouTube embeds via iframe.
- **Hash routing:** Each accordion has `id`. JS auto-opens + scrolls on hash URL arrival.
- **Font:** Inter (Google Fonts, weights 400/500/600).
- **Colors:** White bg, dark text (#333), muted (#888), links (#111).
- **No build step.** Plain HTML/CSS served statically.
